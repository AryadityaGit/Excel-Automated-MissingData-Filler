**Excel-Automated-MissingData-Filler**

An Excel VBA-based framework to automatically fill missing data from historical workbooks.

# Features inside:

* Control Panel sheet (dashboard) for configuration
  1. Folder path, file name filter, lookup & fill columns, options, last run time/date
  2. “Run Macro” button
  3. Auto-colored editable/non-editable cells

* Macro module with:
 1. Configurable columns (default H/I)
 2. Optional file name filter (BOT, NEFT, etc.)
 3. Timer and completion message
 4. Safe formatting (ASCII only, no font lock)
 5. Header tagline and author credit
